## 快速部署指南

### Docker部署（推荐）

```bash
# 1. 配置环境变量
cp .env.example .env
# 编辑 .env 文件，设置 OLLAMA_HOST 为你的GPU服务器地址

# 2. 启动服务
docker-compose up -d

# 3. 查看日志
docker-compose logs -f
```
